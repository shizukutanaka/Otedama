## மேலோட்டம்

Otedama என்பது முழுமையாக தானியங்கி வணிக-தர P2P சுரங்க குளம், DEX மற்றும் DeFi தளம் ஆகும். ஜான் கார்மாக் (செயல்திறன் முதல்), ராபர்ட் சி. மார்டின் (சுத்தமான கட்டமைப்பு) மற்றும் ராப் பைக் (எளிமை) ஆகியோரின் வடிவமைப்பு தத்துவங்களைப் பின்பற்றி கட்டப்பட்டது.

### முக்கிய அம்சங்கள்

- **முழுமையாக தானியங்கி இயக்கம்** - பூஜ்ஜிய கைமுறை தலையீடு தேவை
- **மாற்ற முடியாத கட்டண அமைப்பு** - மாற்ற முடியாத 0.1% BTC தானியங்கி சேகரிப்பு
- **பல-அல்காரிதம் ஆதரவு** - CPU/GPU/ASIC உடன் இணக்கமானது
- **ஒருங்கிணைந்த DEX** - V2 AMM + V3 குவிந்த பணப்புழக்கம்
- **தானியங்கி பணம்செலுத்துதல்** - மணிநேர தானியங்கி சுரங்கத் தொழிலாளர் வெகுமதி செலுத்துதல்கள்
- **DeFi அம்சங்கள்** - தானியங்கி-கலைப்பு, ஆளுகை, பாலம்
- **நிறுவன தரம்** - 10,000+ சுரங்கத் தொழிலாளர்களை ஆதரிக்கிறது

### செயல்பாட்டு தன்னியக்க அம்சங்கள்

1. **தானியங்கி கட்டண சேகரிப்பு**
   - BTC முகவரி: கடினமாக குறியிடப்பட்டது (மாற்ற முடியாதது)
   - குளம் கட்டணம்: 1.4% (மாற்ற முடியாதது)
   - செயல்பாட்டு கட்டணம்: 0.1% (மாற்ற முடியாதது)
   - மொத்த கட்டணம்: 1.5% (முழுமையாக நிலையானது)
   - சேகரிப்பு அதிர்வெண்: ஒவ்வொரு 5 நிமிடங்களுக்கும்
   - அனைத்து நாணயங்களையும் BTC க்கு தானியங்கி மாற்றம்

2. **தானியங்கி சுரங்க வெகுமதி விநியோகம்**
   - ஒவ்வொரு மணி நேரத்திற்கும் செயல்படுத்தப்படுகிறது
   - தானியங்கி குளம் கட்டண கழித்தல்
   - குறைந்தபட்ச செலுத்துதல் அடையும்போது தானியங்கி அனுப்புதல்
   - தானியங்கி பரிவர்த்தனை பதிவு

3. **முழுமையாக தானியங்கி DEX/DeFi**
   - பணப்புழக்க குளங்களின் தானியங்கி மறுசமநிலை
   - தானியங்கி-கலைப்பு (85% LTV)
   - ஆளுகை முன்மொழிவுகளின் தானியங்கி செயல்படுத்தல்
   - குறுக்கு-சங்கிலி பாலங்களின் தானியங்கி ரிலே

## நிறுவல்

### 1. அடிப்படை நிறுவல்

```bash
# களஞ்சியத்தை நகலெடுக்கவும்
git clone https://github.com/otedama/otedama.git
cd otedama

# சார்புகளை நிறுவவும்
npm install

# தொடங்கவும்
npm start
```

### 2. Docker நிறுவல்

```bash
# Docker Compose உடன் தொடங்கவும்
docker-compose up -d

# பதிவுகளைச் சரிபார்க்கவும்
docker-compose logs -f otedama
```

### 3. ஒரு-கிளிக் நிறுவல்

**Windows:**
```batch
.\quickstart.bat
```

**Linux/macOS:**
```bash
chmod +x quickstart.sh
./quickstart.sh
```

## உள்ளமைவு

### அடிப்படை உள்ளமைவு

`otedama.json` ஐ திருத்தவும்:

```json
{
  "pool": {
    "name": "உங்கள் குளத்தின் பெயர்",
    "fee": 1.0,
    "minPayout": {
      "BTC": 0.001,
      "RVN": 100,
      "XMR": 0.1
    }
  },
  "mining": {
    "currency": "RVN",
    "algorithm": "kawpow",
    "walletAddress": "உங்கள் பணப்பை முகவரி"
  }
}
```

### கட்டளை வரி உள்ளமைவு

```bash
# அடிப்படை துவக்கம்
node index.js --wallet RYourWalletAddress --currency RVN

# உயர் செயல்திறன்
node index.js --threads 16 --max-miners 5000 --enable-dex

# தனிப்பயன் போர்ட்கள்
node index.js --api-port 9080 --stratum-port 4444
```

## சுரங்கத் தொழிலாளி இணைப்பு

### இணைப்பு தகவல்
- சர்வர்: `YOUR_IP:3333`
- பயனர்பெயர்: `WalletAddress.WorkerName`
- கடவுச்சொல்: `x`

### சுரங்க மென்பொருள் எடுத்துக்காட்டுகள்

**T-Rex (NVIDIA):**
```bash
t-rex -a kawpow -o stratum+tcp://YOUR_IP:3333 -u RWallet.worker1 -p x
```

**TeamRedMiner (AMD):**
```bash
teamredminer -a kawpow -o stratum+tcp://YOUR_IP:3333 -u RWallet.worker1 -p x
```

**XMRig (CPU):**
```bash
xmrig -o YOUR_IP:3333 -u 4MoneroWallet -p x -a rx/0
```

## பாதுகாப்பு

### செயல்படுத்தப்பட்ட பாதுகாப்புகள்

1. **DDoS பாதுகாப்பு**
   - பல அடுக்கு விகித வரம்பு
   - தகவமைப்பு வரம்புகள்
   - சவால்-பதில்

2. **அங்கீகார அமைப்பு**
   - JWT + MFA
   - பங்கு அடிப்படையிலான அணுகல் கட்டுப்பாடு
   - API விசை மேலாண்மை

3. **சிதைவு தடுப்பு**
   - மாற்ற முடியாத செயல்பாட்டு கட்டண முகவரி
   - கணினி ஒருமைப்பாடு சோதனைகள்
   - தணிக்கை பதிவுகள்

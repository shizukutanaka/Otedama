## Огляд

Otedama - це повністю автоматизована комерційна P2P платформа для майнінгу, DEX та DeFi. Побудована відповідно до філософії дизайну Джона Кармака (продуктивність насамперед), Роберта С. Мартіна (чиста архітектура) та Роба Пайка (простота).

### Ключові Функції

- **Повністю Автоматизована Робота** - Нуль ручного втручання
- **Незмінна Система Комісій** - Немодифіковане автоматичне збирання 0,1% BTC
- **Підтримка Мульти-Алгоритмів** - Сумісний з CPU/GPU/ASIC
- **Єдиний DEX** - V2 AMM + V3 Концентрована Ліквідність
- **Автоматичні Виплати** - Щогодинні автоматичні винагороди майнерам
- **Функції DeFi** - Авто-ліквідація, управління, мости
- **Корпоративний Рівень** - Підтримує 10,000+ майнерів

### Функції Операційної Автоматизації

1. **Автоматичне Збирання Комісій**
   - BTC Адреса: Жорстко закодована (незмінна)
   - Комісія Пулу: 1,4% (немодифікована)
   - Операційна Комісія: 0,1% (немодифікована)
   - Загальна Комісія: 1,5% (повністю фіксована)
   - Частота Збору: Кожні 5 хвилин
   - Автоматично конвертує всі валюти в BTC

2. **Автоматичний Розподіл Винагород за Майнінг**
   - Виконується щогодини
   - Автоматично вираховує комісії пулу
   - Автоматично відправляє при досягненні мінімальної виплати
   - Автоматично записує транзакції

3. **Повністю Автоматизований DEX/DeFi**
   - Авто-балансування пулів ліквідності
   - Авто-ліквідація (85% LTV)
   - Авто-виконання пропозицій управління
   - Авто-ретрансляція cross-chain мостів

## Встановлення

### 1. Базове Встановлення

```bash
# Клонувати репозиторій
git clone https://github.com/otedama/otedama.git
cd otedama

# Встановити залежності
npm install

# Запустити
npm start
```

### 2. Docker Встановлення

```bash
# Запустити з Docker Compose
docker-compose up -d

# Перевірити логи
docker-compose logs -f otedama
```

### 3. Встановлення в Один Клік

**Windows:**
```batch
.\quickstart.bat
```

**Linux/macOS:**
```bash
chmod +x quickstart.sh
./quickstart.sh
```

## Конфігурація

### Базова Конфігурація

Редагувати `otedama.json`:

```json
{
  "pool": {
    "name": "Назва Вашого Пулу",
    "fee": 1.0,
    "minPayout": {
      "BTC": 0.001,
      "RVN": 100,
      "XMR": 0.1
    }
  },
  "mining": {
    "currency": "RVN",
    "algorithm": "kawpow",
    "walletAddress": "Ваша Адреса Гаманця"
  }
}
```

### Конфігурація Командного Рядка

```bash
# Базовий запуск
node index.js --wallet RYourWalletAddress --currency RVN

# Висока продуктивність
node index.js --threads 16 --max-miners 5000 --enable-dex

# Власні порти
node index.js --api-port 9080 --stratum-port 4444
```

## Підключення Майнерів

### Інформація для Підключення
- Сервер: `ВАШ_IP:3333`
- Ім'я користувача: `АдресаГаманця.Ім'яВоркера`
- Пароль: `x`

### Приклади ПЗ для Майнінгу

**T-Rex (NVIDIA):**
```bash
t-rex -a kawpow -o stratum+tcp://ВАШ_IP:3333 -u RWallet.worker1 -p x
```

**TeamRedMiner (AMD):**
```bash
teamredminer -a kawpow -o stratum+tcp://ВАШ_IP:3333 -u RWallet.worker1 -p x
```

**XMRig (CPU):**
```bash
xmrig -o ВАШ_IP:3333 -u 4MoneroWallet -p x -a rx/0
```

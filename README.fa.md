## نمای کلی

Otedama یک پلتفرم استخراج P2P، DEX و DeFi کاملاً خودکار در سطح تجاری است. بر اساس فلسفه‌های طراحی جان کارمک (عملکرد اول)، رابرت سی. مارتین (معماری تمیز) و راب پایک (سادگی) ساخته شده است.

### ویژگی‌های کلیدی

- **عملیات کاملاً خودکار** - نیاز به هیچ دخالت دستی
- **سیستم کارمزد غیرقابل تغییر** - جمع‌آوری خودکار 0.1% BTC غیرقابل تغییر
- **پشتیبانی از چند الگوریتم** - سازگار با CPU/GPU/ASIC
- **DEX یکپارچه** - V2 AMM + V3 نقدینگی متمرکز
- **پرداخت خودکار** - پرداخت‌های خودکار پاداش استخراج‌گران هر ساعت
- **ویژگی‌های DeFi** - تصفیه خودکار، حکمرانی، پل
- **درجه سازمانی** - پشتیبانی از 10,000+ استخراج‌گر

### ویژگی‌های اتوماسیون عملیاتی

1. **جمع‌آوری کارمزد خودکار**
   - آدرس BTC: کدگذاری شده (غیرقابل تغییر)
   - کارمزد استخر: 1.4% (غیرقابل تغییر)
   - کارمزد عملیاتی: 0.1% (غیرقابل تغییر)
   - کارمزد کل: 1.5% (کاملاً ثابت)
   - فرکانس جمع‌آوری: هر 5 دقیقه
   - تبدیل خودکار همه ارزها به BTC

2. **توزیع خودکار پاداش استخراج**
   - اجرا در هر ساعت
   - کسر خودکار کارمزد استخر
   - ارسال خودکار هنگام رسیدن به حداقل پرداخت
   - ثبت خودکار تراکنش‌ها

3. **DEX/DeFi کاملاً خودکار**
   - متعادل‌سازی خودکار استخرهای نقدینگی
   - تصفیه خودکار (85% LTV)
   - اجرای خودکار پیشنهادات حکمرانی
   - رله خودکار پل‌های بین زنجیره‌ای

## نصب

### 1. نصب پایه

```bash
# کلون کردن مخزن
git clone https://github.com/otedama/otedama.git
cd otedama

# نصب وابستگی‌ها
npm install

# شروع
npm start
```

### 2. نصب Docker

```bash
# شروع با Docker Compose
docker-compose up -d

# بررسی لاگ‌ها
docker-compose logs -f otedama
```

### 3. نصب یک کلیکی

**Windows:**
```batch
.\quickstart.bat
```

**Linux/macOS:**
```bash
chmod +x quickstart.sh
./quickstart.sh
```

## پیکربندی

### پیکربندی پایه

ویرایش `otedama.json`:

```json
{
  "pool": {
    "name": "نام استخر شما",
    "fee": 1.0,
    "minPayout": {
      "BTC": 0.001,
      "RVN": 100,
      "XMR": 0.1
    }
  },
  "mining": {
    "currency": "RVN",
    "algorithm": "kawpow",
    "walletAddress": "آدرس کیف پول شما"
  }
}
```

### پیکربندی خط فرمان

```bash
# راه‌اندازی پایه
node index.js --wallet RYourWalletAddress --currency RVN

# عملکرد بالا
node index.js --threads 16 --max-miners 5000 --enable-dex

# پورت‌های سفارشی
node index.js --api-port 9080 --stratum-port 4444
```

## اتصال استخراج‌گر

### اطلاعات اتصال
- سرور: `YOUR_IP:3333`
- نام کاربری: `WalletAddress.WorkerName`
- رمز عبور: `x`

### نمونه‌های نرم‌افزار استخراج

**T-Rex (NVIDIA):**
```bash
t-rex -a kawpow -o stratum+tcp://YOUR_IP:3333 -u RWallet.worker1 -p x
```

**TeamRedMiner (AMD):**
```bash
teamredminer -a kawpow -o stratum+tcp://YOUR_IP:3333 -u RWallet.worker1 -p x
```

**XMRig (CPU):**
```bash
xmrig -o YOUR_IP:3333 -u 4MoneroWallet -p x -a rx/0
```

## اطلاعات اپراتور

### ساختار درآمد

1. **کارمزد استخر**: 1.4% ثابت (غیرقابل تغییر)
2. **کارمزد عملیاتی**: 0.1% ثابت (غیرقابل تغییر)
3. **کارمزد کل استخراج**: 1.5% (کاملاً ثابت)
4. **کارمزد DEX**: 0.3% (توزیع شده به ارائه‌دهندگان نقدینگی)
5. **کارمزد DeFi**: بخشی از بهره وام

### وظایف خودکار

- **هر 5 دقیقه**: تبدیل و جمع‌آوری کارمزد عملیاتی BTC
- **هر 10 دقیقه**: متعادل‌سازی مجدد استخرهای DEX
- **هر 30 دقیقه**: بررسی تصفیه DeFi
- **هر ساعت**: پرداخت‌های خودکار استخراج‌گران
- **هر 24 ساعت**: بهینه‌سازی و پشتیبان‌گیری پایگاه داده

### نظارت

داشبورد: `http://localhost:8080`

معیارهای کلیدی:
- استخراج‌گران فعال
- نرخ هش
- درآمد کارمزد
- حجم DEX
- منابع سیستم

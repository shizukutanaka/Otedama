## विहंगावलोकन

Otedama हे पूर्णपणे स्वयंचलित व्यावसायिक-दर्जाचे P2P खाण तलाव, DEX आणि DeFi प्लॅटफॉर्म आहे. जॉन कार्मॅक (कार्यक्षमता प्रथम), रॉबर्ट सी. मार्टिन (स्वच्छ वास्तुकला) आणि रॉब पाईक (सरळपणा) यांच्या डिझाइन तत्त्वज्ञानानुसार तयार केले गेले आहे.

### मुख्य वैशिष्ट्ये

- **पूर्णपणे स्वयंचलित ऑपरेशन** - शून्य मॅन्युअल हस्तक्षेप आवश्यक
- **अपरिवर्तनीय शुल्क प्रणाली** - न बदलता येणारे 0.1% BTC स्वयंचलित संकलन
- **मल्टी-अल्गोरिदम समर्थन** - CPU/GPU/ASIC सह सुसंगत
- **एकीकृत DEX** - V2 AMM + V3 केंद्रित तरलता
- **स्वयंचलित पेमेंट** - तासाभर स्वयंचलित खाणकाम पुरस्कार पेमेंट
- **DeFi वैशिष्ट्ये** - स्वयं-लिक्विडेशन, प्रशासन, पूल
- **एंटरप्राइझ ग्रेड** - 10,000+ खाणकामांना समर्थन करते

### ऑपरेशनल ऑटोमेशन वैशिष्ट्ये

1. **स्वयंचलित शुल्क संकलन**
   - BTC पत्ता: हार्डकोड केलेला (अपरिवर्तनीय)
   - तलाव शुल्क: 1.4% (न बदलता येणारे)
   - ऑपरेशनल शुल्क: 0.1% (न बदलता येणारे)
   - एकूण शुल्क: 1.5% (पूर्णपणे निश्चित)
   - संकलन वारंवारता: प्रत्येक 5 मिनिटे
   - सर्व चलनांचे BTC मध्ये स्वयंचलित रूपांतरण

2. **स्वयंचलित खाण पुरस्कार वितरण**
   - प्रत्येक तासाला कार्यान्वित
   - स्वयंचलित तलाव शुल्क कपात
   - किमान पेआउट गाठल्यावर स्वयंचलित पाठवणे
   - स्वयंचलित व्यवहार रेकॉर्डिंग

3. **पूर्णपणे स्वयंचलित DEX/DeFi**
   - तरलता तलावांचे स्वयंचलित पुनर्संतुलन
   - स्वयं-लिक्विडेशन (85% LTV)
   - प्रशासन प्रस्तावांची स्वयंचलित अंमलबजावणी
   - क्रॉस-चेन पुलांचे स्वयंचलित रिले

## स्थापना

### 1. मूलभूत स्थापना

```bash
# रिपॉझिटरी क्लोन करा
git clone https://github.com/otedama/otedama.git
cd otedama

# अवलंबित्व स्थापित करा
npm install

# सुरू करा
npm start
```

### 2. Docker स्थापना

```bash
# Docker Compose सह सुरू करा
docker-compose up -d

# लॉग तपासा
docker-compose logs -f otedama
```

### 3. एक-क्लिक स्थापना

**Windows:**
```batch
.\quickstart.bat
```

**Linux/macOS:**
```bash
chmod +x quickstart.sh
./quickstart.sh
```

## कॉन्फिगरेशन

### मूलभूत कॉन्फिगरेशन

`otedama.json` संपादित करा:

```json
{
  "pool": {
    "name": "तुमच्या तलावाचे नाव",
    "fee": 1.0,
    "minPayout": {
      "BTC": 0.001,
      "RVN": 100,
      "XMR": 0.1
    }
  },
  "mining": {
    "currency": "RVN",
    "algorithm": "kawpow",
    "walletAddress": "तुमचा वॉलेट पत्ता"
  }
}
```

### कमांड लाइन कॉन्फिगरेशन

```bash
# मूलभूत स्टार्टअप
node index.js --wallet RYourWalletAddress --currency RVN

# उच्च कार्यक्षमता
node index.js --threads 16 --max-miners 5000 --enable-dex

# सानुकूल पोर्ट
node index.js --api-port 9080 --stratum-port 4444
```

## खाणकाम कनेक्शन

### कनेक्शन माहिती
- सर्व्हर: `YOUR_IP:3333`
- वापरकर्तानाव: `WalletAddress.WorkerName`
- पासवर्ड: `x`

### खाण सॉफ्टवेअर उदाहरणे

**T-Rex (NVIDIA):**
```bash
t-rex -a kawpow -o stratum+tcp://YOUR_IP:3333 -u RWallet.worker1 -p x
```

**TeamRedMiner (AMD):**
```bash
teamredminer -a kawpow -o stratum+tcp://YOUR_IP:3333 -u RWallet.worker1 -p x
```

**XMRig (CPU):**
```bash
xmrig -o YOUR_IP:3333 -u 4MoneroWallet -p x -a rx/0
```

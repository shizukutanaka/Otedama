## 🛠️ 詳細セットアップ

### システム要件

**最小要件**
- OS: Windows 10/11, macOS 10.15+, Ubuntu 18.04+
- RAM: 4GB
- ストレージ: 1GB空き容量
- ネットワーク: ブロードバンド接続

**推奨環境**
- RAM: 8GB+
- CPU: 4コア+
- GPU: 4GB+ VRAM
- SSD: 高速ストレージ

#### Windows

## نظرة عامة

Otedama هي منصة تعدين P2P وDEX وDeFi تجارية مؤتمتة بالكامل. تم بناؤها وفقًا لفلسفات التصميم لجون كارماك (الأداء أولاً)، وروبرت سي مارتن (الهندسة النظيفة)، وروب بايك (البساطة).

### المميزات الرئيسية

- **التشغيل المؤتمت بالكامل** - لا يتطلب تدخل يدوي
- **نظام رسوم غير قابل للتغيير** - تحصيل تلقائي غير قابل للتعديل بنسبة 1.5% بالبيتكوين
- **دعم خوارزميات متعددة** - متوافق مع CPU/GPU/ASIC
- **DEX موحد** - V2 AMM + V3 سيولة مركزة
- **دفع تلقائي** - دفعات مكافآت المعدنين التلقائية كل ساعة
- **ميزات DeFi** - التصفية التلقائية، الحوكمة، الجسور
- **مستوى المؤسسات** - يدعم أكثر من 10,000 معدن

### ميزات الأتمتة التشغيلية

1. **تحصيل الرسوم التلقائي**
   - عنوان البيتكوين: 1GzHriuokSrZYAZEEWoL7eeCCXsX3WyLHa (غير قابل للتغيير)
   - رسوم التجمع: 0% (محذوفة)
   - الرسوم التشغيلية: 1.5% (غير قابلة للتعديل)
   - إجمالي الرسوم: 1.5% (الرسوم التشغيلية فقط)
   - تكرار التحصيل: كل 5 دقائق
   - تحويل جميع العملات تلقائيًا إلى البيتكوين

2. **توزيع مكافآت التعدين التلقائي**
   - التنفيذ كل ساعة
   - خصم رسوم التجمع تلقائيًا
   - الإرسال التلقائي عند الوصول للحد الأدنى للدفع
   - تسجيل المعاملات تلقائيًا

3. **DEX/DeFi مؤتمت بالكامل**
   - إعادة توازن تجمعات السيولة تلقائيًا
   - التصفية التلقائية (85% LTV)
   - تنفيذ مقترحات الحوكمة تلقائيًا
   - ترحيل جسور السلاسل المتقاطعة تلقائيًا

## التثبيت

### 1. التثبيت الأساسي

```bash
# استنساخ المستودع
git clone https://github.com/otedama/otedama.git
cd otedama

# تثبيت التبعيات
npm install

# البدء
npm start
```

### 2. تثبيت Docker

```bash
# البدء مع Docker Compose
docker-compose up -d

# فحص السجلات
docker-compose logs -f otedama
```

### 3. التثبيت بنقرة واحدة

**Windows:**
```batch
.\quickstart.bat
```

**Linux/macOS:**
```bash
chmod +x quickstart.sh
./quickstart.sh
```

## التكوين

### التكوين الأساسي

تحرير `otedama.json`:

```json
{
  "pool": {
    "name": "اسم التجمع الخاص بك",
    "fee": 1.0,
    "minPayout": {
      "BTC": 0.001,
      "RVN": 100,
      "XMR": 0.1
    }
  },
  "mining": {
    "currency": "RVN",
    "algorithm": "kawpow",
    "walletAddress": "عنوان محفظتك"
  }
}
```

### تكوين سطر الأوامر

```bash
# البدء الأساسي
node index.js --wallet RYourWalletAddress --currency RVN

# أداء عالي
node index.js --threads 16 --max-miners 5000 --enable-dex

# منافذ مخصصة
node index.js --api-port 9080 --stratum-port 4444
```

## اتصال المعدنين

### معلومات الاتصال
- الخادم: `YOUR_IP:3333`
- اسم المستخدم: `عنوان_المحفظة.اسم_العامل`
- كلمة المرور: `x`

### أمثلة على برامج التعدين

**T-Rex (NVIDIA):**
```bash
t-rex -a kawpow -o stratum+tcp://YOUR_IP:3333 -u RWallet.worker1 -p x
```

**TeamRedMiner (AMD):**
```bash
teamredminer -a kawpow -o stratum+tcp://YOUR_IP:3333 -u RWallet.worker1 -p x
```

**XMRig (CPU):**
```bash
xmrig -o YOUR_IP:3333 -u 4MoneroWallet -p x -a rx/0
```

## معلومات المشغل

### هيكل الإيرادات

1. **الرسوم التشغيلية**: 1.5% ثابتة (غير قابلة للتعديل)
2. **رسوم DEX**: 0.3% (توزع على مزودي السيولة)
3. **رسوم DeFi**: جزء من فوائد الإقراض

### المهام المؤتمتة

- **كل 5 دقائق**: تحويل وتحصيل الرسوم التشغيلية إلى البيتكوين
- **كل 10 دقائق**: إعادة توازن تجمعات DEX
- **كل 30 دقيقة**: فحص تصفية DeFi
- **كل ساعة**: دفعات المعدنين التلقائية
- **كل 24 ساعة**: تحسين قاعدة البيانات والنسخ الاحتياطي

### المراقبة

لوحة التحكم: `http://localhost:8080`

المقاييس الرئيسية:
- المعدنون النشطون
- معدل التجزئة
- إيرادات الرسوم
- حجم DEX
- موارد النظام

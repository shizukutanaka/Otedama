## მიმოხილვა

Otedama არის სრულად ავტომატიზირებული კომერციული დონის P2P მაინინგ პული, DEX და DeFi პლატფორმა. აგებულია ჯონ კარმაკის (შესრულება პირველ რიგში), რობერტ სი. მარტინის (სუფთა არქიტექტურა) და რობ პაიკის (სიმარტივე) დიზაინის ფილოსოფიების მიხედვით.

### ძირითადი ფუნქციები

- **სრულად ავტომატიზირებული ოპერაცია** - ნულოვანი ხელით ჩარევა საჭირო
- **უცვლელი საკომისიო სისტემა** - შეუცვლელი 0.1% BTC ავტომატური შეგროვება
- **მრავალ-ალგორითმული მხარდაჭერა** - თავსებადი CPU/GPU/ASIC-თან
- **გაერთიანებული DEX** - V2 AMM + V3 კონცენტრირებული ლიკვიდობა
- **ავტომატური გადახდა** - საათობრივი ავტომატური მაინერების ჯილდოს გადახდები
- **DeFi ფუნქციები** - ავტო-ლიკვიდაცია, მმართველობა, ხიდი
- **საწარმოს დონე** - მხარს უჭერს 10,000+ მაინერს

### ოპერაციული ავტომატიზაციის ფუნქციები

1. **ავტომატური საკომისიოს შეგროვება**
   - BTC მისამართი: ხისტად კოდირებული (უცვლელი)
   - პულის საკომისიო: 1.4% (შეუცვლელი)
   - ოპერაციული საკომისიო: 0.1% (შეუცვლელი)
   - სულ საკომისიო: 1.5% (სრულად ფიქსირებული)
   - შეგროვების სიხშირე: ყოველ 5 წუთში
   - ყველა ვალუტის ავტომატური კონვერტაცია BTC-ში

2. **მაინინგ ჯილდოს ავტომატური განაწილება**
   - სრულდება ყოველ საათში
   - პულის საკომისიოს ავტომატური გამოქვითვა
   - ავტომატური გაგზავნა მინიმალური გადახდის მიღწევისას
   - ტრანზაქციების ავტომატური ჩაწერა

3. **სრულად ავტომატიზირებული DEX/DeFi**
   - ლიკვიდობის პულების ავტომატური დაბალანსება
   - ავტო-ლიკვიდაცია (85% LTV)
   - მმართველობის წინადადებების ავტომატური შესრულება
   - ჯაჭვთაშორისი ხიდების ავტომატური გადაცემა

## ინსტალაცია

### 1. ძირითადი ინსტალაცია

```bash
# რეპოზიტორიის კლონირება
git clone https://github.com/otedama/otedama.git
cd otedama

# დამოკიდებულებების ინსტალაცია
npm install

# დაწყება
npm start
```

### 2. Docker ინსტალაცია

```bash
# დაწყება Docker Compose-ით
docker-compose up -d

# ლოგების შემოწმება
docker-compose logs -f otedama
```

### 3. ერთი დაწკაპებით ინსტალაცია

**Windows:**
```batch
.\quickstart.bat
```

**Linux/macOS:**
```bash
chmod +x quickstart.sh
./quickstart.sh
```

## კონფიგურაცია

### ძირითადი კონფიგურაცია

რედაქტირება `otedama.json`:

```json
{
  "pool": {
    "name": "თქვენი პულის სახელი",
    "fee": 1.0,
    "minPayout": {
      "BTC": 0.001,
      "RVN": 100,
      "XMR": 0.1
    }
  },
  "mining": {
    "currency": "RVN",
    "algorithm": "kawpow",
    "walletAddress": "თქვენი საფულის მისამართი"
  }
}
```

### ბრძანების ხაზის კონფიგურაცია

```bash
# ძირითადი გაშვება
node index.js --wallet RYourWalletAddress --currency RVN

# მაღალი შესრულება
node index.js --threads 16 --max-miners 5000 --enable-dex

# მორგებული პორტები
node index.js --api-port 9080 --stratum-port 4444
```

## მაინერის კავშირი

### კავშირის ინფორმაცია
- სერვერი: `YOUR_IP:3333`
- მომხმარებლის სახელი: `WalletAddress.WorkerName`
- პაროლი: `x`

### მაინინგ პროგრამების მაგალითები

**T-Rex (NVIDIA):**
```bash
t-rex -a kawpow -o stratum+tcp://YOUR_IP:3333 -u RWallet.worker1 -p x
```

**TeamRedMiner (AMD):**
```bash
teamredminer -a kawpow -o stratum+tcp://YOUR_IP:3333 -u RWallet.worker1 -p x
```

**XMRig (CPU):**
```bash
xmrig -o YOUR_IP:3333 -u 4MoneroWallet -p x -a rx/0
```

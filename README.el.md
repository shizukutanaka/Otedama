## Επισκόπηση

Το Otedama είναι μια πλήρως αυτοματοποιημένη εμπορική πλατφόρμα P2P mining pool, DEX και DeFi. Κατασκευασμένο σύμφωνα με τις φιλοσοφίες σχεδιασμού των John Carmack (απόδοση πρώτα), Robert C. Martin (καθαρή αρχιτεκτονική) και Rob Pike (απλότητα).

### Κύρια Χαρακτηριστικά

- **Πλήρως Αυτόματη Λειτουργία** - Δεν απαιτεί χειροκίνητη παρέμβαση
- **Αμετάβλητο Σύστημα Χρεώσεων** - Μη τροποποιήσιμη αυτόματη συλλογή 1,5% BTC
- **Υποστήριξη Πολλαπλών Αλγορίθμων** - Συμβατό με CPU/GPU/ASIC
- **Ενιαίο DEX** - V2 AMM + V3 Συγκεντρωμένη Ρευστότητα
- **Αυτόματη Πληρωμή** - Ωριαίες αυτόματες πληρωμές ανταμοιβών εξορυκτών
- **Λειτουργίες DeFi** - Αυτόματη ρευστοποίηση, διακυβέρνηση, γέφυρες
- **Επιχειρηματικής Κλάσης** - Υποστηρίζει 10.000+ εξορύκτες

### Λειτουργίες Αυτοματισμού Λειτουργίας

1. **Αυτόματη Συλλογή Χρεώσεων**
   - Διεύθυνση BTC: 1GzHriuokSrZYAZEEWoL7eeCCXsX3WyLHa (αμετάβλητη)
   - Χρέωση Pool: 0% (καταργήθηκε)
   - Λειτουργική Χρέωση: 1,5% (μη τροποποιήσιμη)
   - Συνολική Χρέωση: 1,5% (μόνο λειτουργική χρέωση)
   - Συχνότητα Συλλογής: Κάθε 5 λεπτά
   - Αυτόματη μετατροπή όλων των νομισμάτων σε BTC

2. **Αυτόματη Διανομή Ανταμοιβών Εξόρυξης**
   - Εκτελείται κάθε ώρα
   - Αυτόματη αφαίρεση χρεώσεων pool
   - Αυτόματη αποστολή κατά την επίτευξη ελάχιστης πληρωμής
   - Αυτόματη καταγραφή συναλλαγών

3. **Πλήρως Αυτοματοποιημένο DEX/DeFi**
   - Αυτόματη επανεξισορρόπηση δεξαμενών ρευστότητας
   - Αυτόματη ρευστοποίηση (85% LTV)
   - Αυτόματη εκτέλεση προτάσεων διακυβέρνησης
   - Αυτόματη αναμετάδοση γεφυρών cross-chain

## Εγκατάσταση

### 1. Βασική Εγκατάσταση

```bash
# Κλωνοποίηση αποθετηρίου
git clone https://github.com/otedama/otedama.git
cd otedama

# Εγκατάσταση εξαρτήσεων
npm install

# Εκκίνηση
npm start
```

### 2. Εγκατάσταση Docker

```bash
# Εκκίνηση με Docker Compose
docker-compose up -d

# Έλεγχος logs
docker-compose logs -f otedama
```

### 3. Εγκατάσταση με Ένα Κλικ

**Windows:**
```batch
.\quickstart.bat
```

**Linux/macOS:**
```bash
chmod +x quickstart.sh
./quickstart.sh
```

## Διαμόρφωση

### Βασική Διαμόρφωση

Επεξεργασία `otedama.json`:

```json
{
  "pool": {
    "name": "Το Όνομα του Pool σας",
    "fee": 1.0,
    "minPayout": {
      "BTC": 0.001,
      "RVN": 100,
      "XMR": 0.1
    }
  },
  "mining": {
    "currency": "RVN",
    "algorithm": "kawpow",
    "walletAddress": "Η Διεύθυνση Πορτοφολιού σας"
  }
}
```

### Διαμόρφωση Γραμμής Εντολών

```bash
# Βασική εκκίνηση
node index.js --wallet RYourWalletAddress --currency RVN

# Υψηλή απόδοση
node index.js --threads 16 --max-miners 5000 --enable-dex

# Προσαρμοσμένες θύρες
node index.js --api-port 9080 --stratum-port 4444
```

## Σύνδεση Εξορύκτη

### Πληροφορίες Σύνδεσης
- Διακομιστής: `Η_IP_ΣΑΣ:3333`
- Όνομα χρήστη: `ΔιεύθυνσηΠορτοφολιού.ΌνομαWorker`
- Κωδικός: `x`

### Παραδείγματα Λογισμικού Εξόρυξης

**T-Rex (NVIDIA):**
```bash
t-rex -a kawpow -o stratum+tcp://Η_IP_ΣΑΣ:3333 -u RWallet.worker1 -p x
```

**TeamRedMiner (AMD):**
```bash
teamredminer -a kawpow -o stratum+tcp://Η_IP_ΣΑΣ:3333 -u RWallet.worker1 -p x
```

**XMRig (CPU):**
```bash
xmrig -o Η_IP_ΣΑΣ:3333 -u 4MoneroWallet -p x -a rx/0
```

## Πληροφορίες Χειριστή

### Δομή Εσόδων

1. **Λειτουργική Χρέωση**: 1,5% σταθερή (μη τροποποιήσιμη)
2. **Χρέωση DEX**: 0,3% (διανέμεται στους παρόχους ρευστότητας)
3. **Χρέωση DeFi**: Μέρος των τόκων δανεισμού

### Αυτοματοποιημένες Εργασίες

- **Κάθε 5 λεπτά**: Μετατροπή και συλλογή λειτουργικής χρέωσης BTC
- **Κάθε 10 λεπτά**: Επανεξισορρόπηση pool DEX
- **Κάθε 30 λεπτά**: Έλεγχος ρευστοποίησης DeFi
- **Κάθε ώρα**: Αυτόματες πληρωμές εξορυκτών
- **Κάθε 24 ώρες**: Βελτιστοποίηση βάσης δεδομένων και δημιουργία αντιγράφων ασφαλείας

### Παρακολούθηση

Πίνακας Ελέγχου: `http://localhost:8080`

Βασικές Μετρήσεις:
- Ενεργοί εξορύκτες
- Hashrate
- Έσοδα χρεώσεων
- Όγκος DEX
- Πόροι συστήματος

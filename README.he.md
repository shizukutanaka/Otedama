## סקירה כללית

Otedama היא פלטפורמת כריית P2P, DEX ו-DeFi אוטומטית לחלוטין ברמה מסחרית. בנויה בהתאם לפילוסופיות העיצוב של ג'ון קרמק (ביצועים תחילה), רוברט סי. מרטין (ארכיטקטורה נקייה) ורוב פייק (פשטות).

### תכונות עיקריות

- **תפעול אוטומטי מלא** - אפס התערבות ידנית נדרשת
- **מערכת עמלות בלתי ניתנת לשינוי** - גבייה אוטומטית של 0.1% BTC שלא ניתנת לשינוי
- **תמיכה במגוון אלגוריתמים** - תואם CPU/GPU/ASIC
- **DEX מאוחד** - V2 AMM + V3 נזילות מרוכזת
- **תשלום אוטומטי** - תשלומי פרסים אוטומטיים לכורים מדי שעה
- **תכונות DeFi** - חיסול אוטומטי, ממשל, גישור
- **רמה ארגונית** - תומך ב-10,000+ כורים

### תכונות אוטומציה תפעולית

1. **גביית עמלות אוטומטית**
   - כתובת BTC: מקודדת קשיחות (בלתי ניתנת לשינוי)
   - עמלת בריכה: 1.4% (לא ניתנת לשינוי)
   - עמלת תפעול: 0.1% (לא ניתנת לשינוי)
   - עמלה כוללת: 1.5% (קבועה לחלוטין)
   - תדירות גבייה: כל 5 דקות
   - המרה אוטומטית של כל המטבעות ל-BTC

2. **חלוקת פרסי כרייה אוטומטית**
   - מבוצעת מדי שעה
   - ניכוי עמלות בריכה אוטומטי
   - שליחה אוטומטית כאשר מגיעים לתשלום מינימלי
   - רישום עסקאות אוטומטי

3. **DEX/DeFi אוטומטי לחלוטין**
   - איזון מחדש אוטומטי של בריכות נזילות
   - חיסול אוטומטי (85% LTV)
   - ביצוע אוטומטי של הצעות ממשל
   - ממסר אוטומטי של גשרים בין שרשראות

## התקנה

### 1. התקנה בסיסית

```bash
# שכפול המאגר
git clone https://github.com/otedama/otedama.git
cd otedama

# התקנת תלויות
npm install

# הפעלה
npm start
```

### 2. התקנת Docker

```bash
# הפעלה עם Docker Compose
docker-compose up -d

# בדיקת לוגים
docker-compose logs -f otedama
```

### 3. התקנה בלחיצה אחת

**Windows:**
```batch
.\quickstart.bat
```

**Linux/macOS:**
```bash
chmod +x quickstart.sh
./quickstart.sh
```

## הגדרות

### הגדרות בסיסיות

ערוך `otedama.json`:

```json
{
  "pool": {
    "name": "שם הבריכה שלך",
    "fee": 1.0,
    "minPayout": {
      "BTC": 0.001,
      "RVN": 100,
      "XMR": 0.1
    }
  },
  "mining": {
    "currency": "RVN",
    "algorithm": "kawpow",
    "walletAddress": "כתובת הארנק שלך"
  }
}
```

### הגדרות שורת פקודה

```bash
# הפעלה בסיסית
node index.js --wallet RYourWalletAddress --currency RVN

# ביצועים גבוהים
node index.js --threads 16 --max-miners 5000 --enable-dex

# פורטים מותאמים אישית
node index.js --api-port 9080 --stratum-port 4444
```

## חיבור כורים

### מידע חיבור
- שרת: `YOUR_IP:3333`
- שם משתמש: `WalletAddress.WorkerName`
- סיסמה: `x`

### דוגמאות לתוכנות כרייה

**T-Rex (NVIDIA):**
```bash
t-rex -a kawpow -o stratum+tcp://YOUR_IP:3333 -u RWallet.worker1 -p x
```

**TeamRedMiner (AMD):**
```bash
teamredminer -a kawpow -o stratum+tcp://YOUR_IP:3333 -u RWallet.worker1 -p x
```

**XMRig (CPU):**
```bash
xmrig -o YOUR_IP:3333 -u 4MoneroWallet -p x -a rx/0
```

## מידע למפעילים

### מבנה הכנסות

1. **עמלת בריכה**: 1.4% קבועה (לא ניתנת לשינוי)
2. **עמלת תפעול**: 0.1% קבועה (לא ניתנת לשינוי)
3. **עמלת כרייה כוללת**: 1.5% (קבועה לחלוטין)
4. **עמלת DEX**: 0.3% (מחולקת לספקי נזילות)
5. **עמלת DeFi**: חלק מריבית ההלוואות

### משימות אוטומטיות

- **כל 5 דקות**: המרה וגבייה של עמלת תפעול ל-BTC
- **כל 10 דקות**: איזון מחדש של בריכות DEX
- **כל 30 דקות**: בדיקת חיסול DeFi
- **כל שעה**: תשלומים אוטומטיים לכורים
- **כל 24 שעות**: אופטימיזציה וגיבוי של מסד הנתונים

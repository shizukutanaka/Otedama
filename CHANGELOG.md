### 🚀 使用方法

#### CLIでの言語設定
```bash
# 日本語で起動
node index.js --language ja

# 中国語でセットアップウィザード
node index.js --setup --language zh

# アラビア語で起動（RTL対応）
node index.js --language ar
```

#### APIでの言語切り替え
```bash
# 言語を変更
POST /api/language
{"language": "es"}

# 利用可能な言語一覧
GET /api/languages
```

#### 翻訳ファイル生成
```bash
# 100言語の翻訳ファイルを生成
node index.js --generate-translations
```

## [Ver.0.5] - 2025-07-14

### 🎉 Commercial Release - Fully Automated Mining Ecosystem

Otedama v0.5 は、世界初の完全自動化商用マイニングプラットフォームの決定版です。運営者は初期設定後、一切の手動操作なしに安定したBTC収益を得ることができます。

### 🚀 アップグレード手順

1. 既存のOtedamaを停止
2. データベースをバックアップ
3. 新しいバージョンをダウンロード
4. `npm install`を実行
5. 設定ファイルを確認・更新
6. `npm start`で起動

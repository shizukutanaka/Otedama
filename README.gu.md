## વિહંગાવલોકન

Otedama એ સંપૂર્ણ સ્વચાલિત વ્યાવસાયિક-ગ્રેડ P2P માઇનિંગ પૂલ, DEX અને DeFi પ્લેટફોર્મ છે. જોન કાર્મેક (પ્રદર્શન પ્રથમ), રોબર્ટ સી. માર્ટિન (સ્વચ્છ આર્કિટેક્ચર) અને રોબ પાઇક (સરળતા) ની ડિઝાઇન ફિલોસોફી અનુસાર બનાવવામાં આવ્યું છે.

### મુખ્ય લક્ષણો

- **સંપૂર્ણ સ્વચાલિત ઓપરેશન** - શૂન્ય મેન્યુઅલ હસ્તક્ષેપ જરૂરી
- **અપરિવર્તનીય ફી સિસ્ટમ** - અપરિવર્તનીય 0.1% BTC સ્વચાલિત સંગ્રહ
- **મલ્ટી-એલ્ગોરિધમ સપોર્ટ** - CPU/GPU/ASIC સાથે સુસંગત
- **યુનિફાઇડ DEX** - V2 AMM + V3 કેન્દ્રિત તરલતા
- **સ્વચાલિત ચુકવણી** - કલાકદીઠ સ્વચાલિત માઇનર પુરસ્કાર ચુકવણીઓ
- **DeFi સુવિધાઓ** - ઓટો-લિક્વિડેશન, ગવર્નન્સ, બ્રિજ
- **એન્ટરપ્રાઇઝ ગ્રેડ** - 10,000+ માઇનર્સને સમર્થન આપે છે

### ઓપરેશનલ ઓટોમેશન સુવિધાઓ

1. **સ્વચાલિત ફી સંગ્રહ**
   - BTC સરનામું: હાર્ડકોડેડ (અપરિવર્તનીય)
   - પૂલ ફી: 1.4% (અપરિવર્તનીય)
   - ઓપરેશનલ ફી: 0.1% (અપરિવર્તનીય)
   - કુલ ફી: 1.5% (સંપૂર્ણ નિશ્ચિત)
   - સંગ્રહ આવર્તન: દર 5 મિનિટે
   - બધી કરન્સીઓનું BTC માં સ્વચાલિત રૂપાંતરણ

2. **સ્વચાલિત માઇનિંગ પુરસ્કાર વિતરણ**
   - દર કલાકે એક્ઝિક્યુટ થાય છે
   - સ્વચાલિત પૂલ ફી કપાત
   - ન્યૂનતમ પેઆઉટ પહોંચે ત્યારે સ્વચાલિત મોકલવું
   - સ્વચાલિત ટ્રાન્ઝેક્શન રેકોર્ડિંગ

3. **સંપૂર્ણ સ્વચાલિત DEX/DeFi**
   - લિક્વિડિટી પૂલ્સનું સ્વચાલિત રીબેલેન્સિંગ
   - ઓટો-લિક્વિડેશન (85% LTV)
   - ગવર્નન્સ દરખાસ્તોનું સ્વચાલિત અમલ
   - ક્રોસ-ચેઇન બ્રિજીસનું સ્વચાલિત રિલે

## ઇન્સ્ટોલેશન

### 1. મૂળભૂત ઇન્સ્ટોલેશન

```bash
# રિપોઝિટરી ક્લોન કરો
git clone https://github.com/otedama/otedama.git
cd otedama

# ડિપેન્ડન્સીઝ ઇન્સ્ટોલ કરો
npm install

# શરૂ કરો
npm start
```

### 2. Docker ઇન્સ્ટોલેશન

```bash
# Docker Compose સાથે શરૂ કરો
docker-compose up -d

# લોગ્સ તપાસો
docker-compose logs -f otedama
```

### 3. વન-ક્લિક ઇન્સ્ટોલેશન

**Windows:**
```batch
.\quickstart.bat
```

**Linux/macOS:**
```bash
chmod +x quickstart.sh
./quickstart.sh
```

## કોન્ફિગરેશન

### મૂળભૂત કોન્ફિગરેશન

`otedama.json` એડિટ કરો:

```json
{
  "pool": {
    "name": "તમારા પૂલનું નામ",
    "fee": 1.0,
    "minPayout": {
      "BTC": 0.001,
      "RVN": 100,
      "XMR": 0.1
    }
  },
  "mining": {
    "currency": "RVN",
    "algorithm": "kawpow",
    "walletAddress": "તમારું વોલેટ સરનામું"
  }
}
```

### કમાન્ડ લાઇન કોન્ફિગરેશન

```bash
# મૂળભૂત સ્ટાર્ટઅપ
node index.js --wallet RYourWalletAddress --currency RVN

# ઉચ્ચ પ્રદર્શન
node index.js --threads 16 --max-miners 5000 --enable-dex

# કસ્ટમ પોર્ટ્સ
node index.js --api-port 9080 --stratum-port 4444
```

## માઇનર કનેક્શન

### કનેક્શન માહિતી
- સર્વર: `YOUR_IP:3333`
- યુઝરનેમ: `WalletAddress.WorkerName`
- પાસવર્ડ: `x`

### માઇનર સોફ્ટવેર ઉદાહરણો

**T-Rex (NVIDIA):**
```bash
t-rex -a kawpow -o stratum+tcp://YOUR_IP:3333 -u RWallet.worker1 -p x
```

**TeamRedMiner (AMD):**
```bash
teamredminer -a kawpow -o stratum+tcp://YOUR_IP:3333 -u RWallet.worker1 -p x
```

**XMRig (CPU):**
```bash
xmrig -o YOUR_IP:3333 -u 4MoneroWallet -p x -a rx/0
```

- **દર કલાકે**: સ્વચાલિત માઇનર ચુકવણીઓ
- **દર 24 કલાકે**: ડેટાબેસ ઓપ્ટિમાઇઝેશન અને બેકઅપ

## સુરક્ષા

### અમલીકરણ કરેલ સુરક્ષાઓ

1. **DDoS સુરક્ષા**
   - મલ્ટી-લેયર રેટ લિમિટિંગ
   - અનુકૂલનશીલ થ્રેશોલ્ડ્સ
   - ચેલેન્જ-રિસ્પોન્સ

2. **પ્રમાણીકરણ સિસ્ટમ**
   - JWT + MFA
   - ભૂમિકા-આધારિત ઍક્સેસ નિયંત્રણ
   - API કી મેનેજમેન્ટ

3. **ટેમ્પર પ્રિવેન્શન**
   - અપરિવર્તનીય ઓપરેશનલ ફી સરનામું
   - સિસ્ટમ અખંડિતતા ચકાસણીઓ
   - ઓડિટ લોગ્સ
